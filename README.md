#  LLM Chat UI (Groq API)

This project is a custom LLM chat interface built using the **free Groq API**.  
It provides a ChatGPT-like experience with support for multiple models, adjustable parameters, and local memory.

---

##  Features

-  **Chat Interface**
  - Real-time streaming responses (like ChatGPT)
  - Enter-to-send support

-  **Multiple Models**
  - Users can switch between different LLM models provided by Groq

-  **Temperature Control**
  - Adjust response randomness using a slider

-  **Custom UI Theme**
  - Select accent color via gradient color picker
  - (Supports future dark mode expansion)

-  **Conversation Management**
  - Multiple conversations
  - Folder organization
  - Delete and manage chats

-  **Local Memory**
  - Chat history is stored in browser `localStorage`
  - No backend database required

---

##  Tech Stack

- **Frontend:** Next.js (App Router) + React + TypeScript
- **Styling:** Tailwind CSS
- **API:** Groq (OpenAI-compatible API)
- **State Storage:** localStorage
